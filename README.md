# Hackeresume
Build and deploy a social network with Node.js, Express, React, Redux & MongoDB

## Tech and learning
- 使用 Node.js ＆ Express 建構廣泛的後端 API
- 使用 JWT（ JSON Web Tokens） 保護路由/端點( endpoints )
- 用 Postman 進行廣泛的 API 測試
- 將 React 與Express後端整合
- 建構我們的前端以和 API 合作
- 使用 Redux 進行應用程式狀態管理
- 為我們的資源創建減速器( reducers )和活動( actions )
- 創建與 Redux 整合的許多容器組合元件
- 使用 Redux Chrome 擴充程式進行測試
- 創建建構腳本、保護我們的密鑰並使用 Git 佈署到 Heroku
